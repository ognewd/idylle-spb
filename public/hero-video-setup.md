# Настройка видео для Hero блока

## Требования к видео

1. **Формат**: MP4 (предпочтительно) и WebM (для лучшей совместимости)
2. **Разрешение**: 1920x1080 (Full HD) или меньше для оптимизации
3. **Длительность**: 10-30 секунд (зацикленное)
4. **Размер файла**: максимум 5-10 МБ для быстрой загрузки
5. **Качество**: можно немного сжать для web

## Как сжать видео

### Использование FFmpeg:

```bash
# Создание оптимизированного MP4
ffmpeg -i input.mp4 -vcodec h264 -acodec mp2 -crf 28 -preset slow -pix_fmt yuv420p -movflags +faststart hero-video.mp4

# Создание WebM версии
ffmpeg -i input.mp4 -c:v libvpx-vp9 -crf 30 -b:v 0 -c:a libopus hero-video.webm

# Создание превью кадра для быстрой загрузки
ffmpeg -i input.mp4 -ss 00:00:01 -vframes 1 hero-video-poster.jpg
```

## Рекомендации по контенту

- **Атмосферные сцены**: создание ароматов, коллекция флаконов
- **Замедленные кадры**: парфюмер наливает духи, процесс создания
- **Абстрактные образы**: цветы, природа, элегантные движения
- **Специальные эффекты**: размытие, движение, изменение фокуса

## Размещение файлов

Поместите файлы в папку `public/`:
```
public/
  ├── hero-video.mp4
  ├── hero-video.webm
  └── hero-video-poster.jpg (опционально)
```

## Оптимизация производительности

Настройки в коде уже оптимизированы:
- ✅ `preload="metadata"` - загружает только метаданные сначала
- ✅ `autoPlay` и `muted` - автовоспроизведение работает в браузерах
- ✅ `loop` - бесконечный цикл
- ✅ `playsInline` - воспроизведение встроено на мобильных
- ✅ Низкая непрозрачность (`opacity-20`) - видео как тонкий фон
- ✅ Градиентный оверлей - сохраняет читаемость текста

## Где взять видео

1. **Бесплатные стоки**:
   - Pexels Videos (pexels.com/videos)
   - Pixabay (pixabay.com/videos)
   - Mixkit (mixkit.co)

2. **Поисковые запросы**:
   - "perfume making", "создание духов"
   - "luxury perfume", "люксовые духи"
   - "fragrance bottles", "флаконы для духов"
   - "slow motion perfume", "замедленная съемка духов"

3. **Создать самостоятельно**:
   - Снять 10-30 секунд атмосферного видео
   - Использовать смартфон с хорошей камерой
   - Спецэффекты можно добавить в мобильных редакторах

## Альтернатива без видео

Если вы не хотите использовать видео, можно удалить блок с `<video>` из `src/app/page.tsx` - градиентный фон останется и будет работать без изменений.
